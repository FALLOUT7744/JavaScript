## 下記for文の`let i = 0`, `i < 100`, `$i++`, `console.log(i)`がそれぞれ、どの順番で処理されるか、また、何をしているのかを説明してください。

```
for (let i = 0; i < 100; i++) {
  console.log(i);
}
```

1. i = 0 : 初期値を設定
2. i < 100 :  条件式（100未満）
3. console.log(i) : 2の条件式と一致していれば処理を実行
4. i++ : 後処理（１を足す）
